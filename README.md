# HowOldAreYou
借着前一段时间微软公司开发的人脸识别测年龄火起来的效应，<br>
利用Face++接口，实现的根据人脸照片识别出性别和年龄，同时在图片上用白色方框标识出人脸。<br>
<br>
测试机型：华为荣耀6
主要外部接口：Face++
